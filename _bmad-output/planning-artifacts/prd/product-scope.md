# Product Scope

To validate this vision, secondbrain is built in three phased releases. MVP validates core value; Phase 2 adds exploitation features; Phase 3+ explores SaaS potential.

## MVP Phase 1 (MVP Core)

**MUST HAVE:**
- Capture interface (markdown input)
- Tier 1 + Tier 2 suggestion system (keywords + Claude API)
- Refinement interface (accept/reject suggestions)
- Search (simple keyword + fuzzy)
- Note history/review
- Weekly digest generation (basic: accomplishments + challenges + next week)

**MUST NOT HAVE (Explicitly Deferred):**
- Advanced search (semantic, conversational)
- Knowledge graphs / visualizations
- iPad/OCR input
- Collaboration/sharing
- Mobile app

## Growth Phase 2 (Post-MVP)

**Adds:**
- Eisenhower matrix visualization
- Advanced search (semantic + AI-powered Q&A)
- Pattern detection and insights
- LLM local integration (when GPU available)

## Vision Phase 3+ (Future)

**Future Potential:**
- Fine-tuned models on your data corpus
- iPad/stylus input + OCR
- Team collaboration (if SaaS path chosen)
- Web clipper / browser integration
